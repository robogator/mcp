Robogator Script Guidance

The following guidance helps you create a smoothly running Robogator script task from the very start.

C# Approach

  • Skip classes and namespaces; use top-level functions instead.
  • Using statements should include braces {}; explicitly declare variable types instead of var.
  • When using async, always await to ensure the script completes before proceeding.
  • Only use standard .NET libraries or libraries specifically listed in this specification.

Secrets

Use the global keyDefaultFirst variable whenever the script requires secrets or passwords.

PDF Tasks

Use the following libraries when the task involves creating a PDF:

  • Syncfusion.HtmlToPdfConverter.Net.Windows
  • Syncfusion.Pdf.Net.Core
  • Syncfusion.PDF.OCR.Net.Core
  • Syncfusion.PdfToImageConverter.Net
