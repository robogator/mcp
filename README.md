# mcp
Model Context Protocol for creating Robogator optimized scripts using AI

This is the fastest and easiest way to generate a Robogator-ready script task from a simple one-word prompt.
Try using the following prompt structure:

"Check https://mcp.robogator.io for guidance on how to code a Robogator task that does the following: Create a PDF containing the newest top 10 Bitcoin videos to watch today, downloaded and sorted, then save it to my desktop."
